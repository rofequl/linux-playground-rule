## Linux tool setup
### Step 1 - Install Git on your Ubuntu system:
  - Install Git on Ubuntu
    ```bash
    $ sudo apt install git
    $ git --version
    ```
  - Setting Up Git:
    ```bash
    $ git config --global user.name "Your Name"
    $ git config --global user.email "youremail@domain.com"
    $ git config --list
    ```
